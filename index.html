TextCase Converter

Text Case Conversion Tool

Data Collection

We do not collect any personal data. All text processing is performed entirely within your browser using local JavaScript. Your text never leaves your device.

Local Storage

The extension uses Chrome's local storage API to save:

  • Your conversion history (text only, stored locally)
  • Your custom presets and preferences
  • Pro license status (if applicable)

All stored data remains on your device and can be cleared at any time through your browser settings.

Permissions

The extension requires minimal permissions:

  • Storage: To save your history and preferences locally
  • Clipboard: To copy converted text to your clipboard (only when you click "Copy")
  • Active Tab: Only when using context menu features

Third-Party Services

This extension does not use any third-party services, analytics, or tracking. There are no external API calls.

Image Watermarker

Image Watermarking Tool

Data Collection

We do not collect any images or personal data. All image processing happens locally in your browser using HTML5 Canvas. Your images are never uploaded to any server.

Local Storage

The extension uses Chrome's local storage API to save:

  • Your watermark settings (text, position, font, opacity, color)
  • Your custom presets
  • Thumbnail previews of your watermarked images
  • Pro license status (if applicable)

All stored data remains on your device. Thumbnails are small base64-encoded previews stored locally.

Permissions

The extension requires minimal permissions:

  • Storage: To save your settings and history locally
  • Downloads: To save watermarked images to your Downloads folder
  • Active Tab: Only when using context menu features

Image Processing

All image processing is done locally using the browser's Canvas API. Your original images and watermarked outputs never leave your device unless you choose to download them.

Third-Party Services

This extension does not use any third-party services, cloud storage, or external APIs. All operations are 100% local.
